How do I play?

Default keys are left/right mouse buttons and Z/X keys, which can be changed in settings. Circles that appear on screen are called hit circles. Click them when the outer approach circle shrinks and overlaps with the hit circle. Some hit circles have sliders attached. Hold down the key and follow the slider path. When a large spinner appears, hold down a key and spin rapidly around the center.

The cursor feels laggy?

Try enabling hardware cursor in settings to reduce cursor latency. However, this may cause flickering or cursor visibility issues on some systems or browsers, so it's disabled by default. Currently, hardware cursor size has only 3 fixed levels: minimum below 0.65, maximum above 0.95.

The game is very laggy, what should I do?

In settings, uncheck "Use system resolution" and set a lower resolution than your system scaling (laptops and high-resolution displays typically have scaling above 100%). Avoid high CPU usage, close antivirus software, try closing other browser tabs and restart the browser. Make sure you're using a modern mainstream browser (Firefox or Chrome). If it's still laggy, please file an issue.

Why is nothing showing on the page?

The mirror site providing beatmaps may be experiencing issues. You can check if the Sayobot homepage also has no content. If so, contact the mirror site administrator for recovery.

My download manager (IDM) keeps showing popups!

This site uses XHR to load audio and visual resources. Please configure your download manager (IDM) to disable it on this site.

This feature is still in development. You can search for beatmaps already in the database.
